## 🤖 フェーズ3: 毎日AI記事生成・自動化ワークフロー（NEW）

Executive Marathon に**毎日自動で記事を生成・投稿する完全自動化パイプライン**を実装します。

### 📋 自動化フロー全体

```
毎朝 6:08 AM (自動トリガー)
    ↓
[AI が記事を自動生成]
  ├ 30 記事アウトラインから次の記事を選択
  ├ Markdown 形式で 800-1500 語を生成
  ├ ビジネス×ランニング視点を反映
  └ フロントマター自動付与
    ↓
[自動品質チェック]
  ├ 文字数、SEO、リンク構造を検証
  └ チェックリストを HTML ダッシュボードに表示
    ↓
[TA (著者) がレビュー・承認]
  ├ ダッシュボードで 5-10 分で確認
  ├ 修正が必要なら Word でトラッキング変更で指摘
  └ 「承認」ボタンをクリック
    ↓
[GitHub に自動コミット]
  ├ ファイル名：YYYYMMDD-article-title.md
  ├ content/blog/ に追加
  └ コミットメッセージ：「Post: Article Title (Category)」
    ↓
[Cloudflare Pages が自動ビルド]
  ├ GitHub webhook トリガー
  ├ Hugo でビルド（30 秒）
  └ CDN に配信
    ↓
[本番サイト更新完了]
  ├ https://executive-marathon.com に新記事が反映
  └ Google Search Console に自動通知（サイトマップ更新）
```
